# Excel Basic

## 구성요소

- Tab menu: 명령 탭
- Reborn menu: 세부 명령
- WorkSeet: 실제 작업하는 부분(행, 열, 셀)

- 셀: 값(숫자, 텍스트), 수식(사칙/논리연산), 서식(폰트, 글자크기 등)

## 자주쓰는 단축키
|키|설명|키|설명|
|:---:|:---:|:---:|:---:|
|F2|셀에 수식/값 입력|Alt + Enter|셀 내에서 줄 넘김|
|F4|마지막 동작 반복|Ctrl + 클릭|다중 선택|
|F12|다른 이름으로 저장|Ctrl + A|전체 선택|
|Ctrl + B|볼드체|Ctrl + Z\Y|이전/이후 작업으로 돌아가기|
|Ctrl + 방향키|값이 있는 마지막 셀로 이동|Ctrl + Alt + V|불여넣기 옵션 실행|
|Shift + 클릭|선택한 셀(1)~(2)까지 모두 선택|Ctrl + page up/down|시트 이동(좌/우)|
|Alt + Shift + 방향키|행/열 그룹 설정/해제|Ctrl + Shift + 방향키|값이 있는 마지막 셀까지 선택|

## 참조 종류

- 상대 참조 : A1
- 절대 참조 : $A$1
- 혼합 참조 : A$1

## 기초통계량

|키|설명|키|설명|
|:---:|:---:|:---:|:---:|
|관측수|=count()|합|=sum()|
|평균|=average()|최빈값|=mode()|
|중앙값|=median()|최대값|=Max or large(범위,1)|
|최소값|=Min or small(범위,1)|분위|=Quartile.INC(범위,0~4)|
|분산|=Var.S()|표준편차|=Stdev.S()|
|여러 조건 만족 값 합|SUMIFS(범위, 조건범위1, 조건1, 조건범위2, 조건2...)|||

## 텍스트 함수

|키|설명|키|설명|
|:---:|:---:|:---:|:---:|
|왼쪽부터 추출|=LEFT(Text, 가져올 텍스트 개수)|오른쪽부터 추출|=RIGHT(Text, 가져올 텍스트 개수)|
|원하는 위치부터 추출|=MID(Text, 시작할 위치, 가져올 텍스트 개수)|텍스트 글자 수 추출|=LEN(Text)|
|특정 문자 순서 찾기|=FIND(찾는문자, 텍스트)|텍스트 병함|=CONCAT(Text1, Text2...)|
|텍스트 서식 지정 함수|=TEXT(값, "표시형식코드")|||

## 조건문

- 논리함수: IF(논리식, 참일 때 값, 거짓일 때 값)
- 논리함수 중첩문: IF(논리식, 참일 때 값, IF(논리식, 참일때 값, IF(...))))...
