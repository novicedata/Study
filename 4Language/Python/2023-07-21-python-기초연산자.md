# 📌 python 비교 연산자

---

아래 코드를 확인해주세요.

```python
a = 5
b = 20

print(a > b) # 크다

print(a >= b) # 크거나 같다

print(a < b) # 작다

print(a <= b) # 작거나 같다

print(a == b) # 같다

print(a != b) #다르다
```

<div class ="notice--primary" markdown="1">
결과
```python
False
False
True
True
False
True
```
</div>

**`이상 python 기초 연산자였습니다! ☠️`**
